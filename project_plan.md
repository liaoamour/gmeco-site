# GM Eco Global - B2B Hospitality Supplies Website

## 1. Project Description
GM Eco Global is a B2B hospitality supplies supplier targeting European hotel clients. The website serves as a sales conversion tool — helping potential buyers quickly understand products, pricing positioning, and reliability, then take action to contact the supplier. The site is NOT a brand story page; it's a direct B2B lead generation tool.

## 2. Page Structure
- `/` - Single-page website with sections:
  - Hero
  - Products (Slippers, Towels & Linen, Pet-Friendly Supplies)
  - Advantages (Why Choose Us)
  - Trust Module
  - Contact & CTA
  - Footer

## 3. Core Features
- [x] Hero section with headline, subheadline, and CTA buttons
- [x] Product showcase (3 product categories)
- [x] Advantages section (5 points)
- [x] Trust/credibility module
- [x] Contact section with email, WhatsApp, website
- [x] Quote request form
- [x] Responsive design (desktop-first)

## 4. Data Model Design
No database needed — static B2B showcase site.

## 5. Backend / Third-party Integration Plan
- Supabase: Not needed
- Shopify: Not needed
- Stripe: Not needed
- Form: Using Readdy form API for quote requests
